# 효도비서 (Hyo-Do Assistant)

## 서비스 개요

**효도비서**는 "어려운 스마트 용무, 이제 채팅 한 통으로 해결하세요"를 슬로건으로 하는 **디지털 심부름** 서비스입니다.

복잡한 앱 조작 없이, 전담 비서(AI 및 상담원)가 기차표 예매부터 병원 동행 예약까지 대행해주는 서비스를 제공합니다.

### 주요 타겟

- **사용자**: 디지털 기기 사용에 피로감을 느끼는 60-80대 액티브 시니어
- **구매자**: 부모님의 일상 업무를 대신해주기 힘든 바쁜 30-50대 직장인 자녀

## 핵심 서비스 기능 (The Solutions)

### ① 대화형 예약·구매 엔진 (Conversational Booking)

- **방식**: 카카오톡 형태의 채팅창 또는 음성 명령 인터페이스
- **내용**: 기차표/버스표 예매, 병원 진료 예약, 골프장 부킹, 인기 식당(캐치테이블 등) 예약 대행
- **특징**: "이번 주 토요일 오후 2시에 대전 가는 기차표 끊어줘"라고 말하면 상담원이 최적의 옵션을 제안하고 결제까지 완료

### ② 행정 및 민원 서류 대행 가이드

- **내용**: 주민등록등본 발급, 건강보험 납부확인서 등 정부24 업무 지원
- **방식**: 복잡한 보안 인증 과정을 비서가 원격으로 안내하거나, 대행 가능한 서류는 위임받아 처리

### ③ 시니어 특화 오프라인 동행 매칭

- **내용**: 단순 예약에서 나아가, 대형병원 진료 시 동행하여 수납과 처방전 수령을 돕는 '동행 매니저' 연결

## 사용자 경험(UX) 설계 전략

- **No-Manual UI**: 매뉴얼이 필요 없는 단순성. 큰 글씨, 고대비 색상, 음성 인식 우선
- **자녀 공유 기능**: 부모님이 예매한 내역(기차 시간, 병원 예약 등)이 자녀에게 실시간 알림으로 전송되어 안심 제공
- **Safe-Payment**: 자녀의 카드를 미리 등록해두면, 부모님이 요청한 건에 대해 자녀가 승인 버튼만 누르면 결제되는 '패밀리 페이' 도입

## 비즈니스 모델 (Revenue Model)

| 모델명 | 상세 내용 | 가격 |
| --- | --- | --- |
| **월간 구독권 (Basic)** | 월 3회 예매 대행, 24시간 전화 상담 지원 | 월 19,900원 |
| **프리미엄 구독 (Gold)** | 무제한 예매 대행, 병원 동행 매칭 할인, 자녀 리포트 제공 | 월 49,900원 |
| **건당 이용료** | 비구독자 대상 단발성 예약 대행 | 건당 3,000 ~ 5,000원 |
| **B2B 파트너십** | 보험사/카드사 고객 대상 제휴 부가 서비스 제공 | 법인별 별도 계약 |

## 운영 전략

### 1단계: 하이브리드 상담 시스템 (AI + Human)

- 단순 검색 및 정보 제공은 LLM(AI)이 처리하여 비용 절감
- 실제 결제 및 복잡한 예약은 전문 교육을 받은 **컨시어지 상담원**이 개입하여 신뢰도 확보

### 2단계: 신뢰 구축 (Trust)

- 개인정보 보안을 위해 모든 상담원 '범죄경력조회' 실시 및 '개인정보 보호 배상책임보험' 가입
- 지자체 시니어 일자리 사업과 연계하여, 건강한 시니어가 더 고령의 시니어를 돕는 '노노(老老)케어' 모델로 확장

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Package Manager**: pnpm
- **Deployment**: Vercel (권장)

## 시작하기

### 필수 요구사항

- Node.js 18.0 이상
- pnpm 10.12.1 이상

### 설치

```bash
pnpm install
```

### 개발 서버 실행

```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
pnpm build
```

### 프로덕션 실행

```bash
pnpm start
```

### 린트

```bash
pnpm lint
```

## 기대 효과 및 미래 확장성

- **시장성**: 디지털 격차가 심해질수록 대행 서비스의 가치는 상승함
- **확장성**: 축적된 데이터(선호 식당, 주로 가는 병원, 여행 패턴)를 바탕으로 **'시니어 타겟 맞춤형 광고'** 및 **'건강기능식품 큐레이션'**으로 비즈니스 확장 가능
